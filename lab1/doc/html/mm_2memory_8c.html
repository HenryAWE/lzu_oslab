◆ put_page()

uint64_t put_page ( uint64_t  page,
uint64_t  addr 
)

put_page()用于建立物理地址和虚拟地址间的映射，物理页必须是 已分配但未映射 的， 不能将未分配或已经映射到别的虚拟页的物理页映射到另一个虚拟页。

当分配物理页失败（创建页表）时 panic,因此不需要检测返回值。

参数
page物理地址
addr虚拟地址
返回
虚拟地址
参见
put_dirty_page(), panic()
